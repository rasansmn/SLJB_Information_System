Sri Lanka Job Bank - Resume and Job Campaign Management - 2013 - Rasan Samarasinghe

Web application developed to manage day to day activities and job seekers information of Sri Lanka Job Bank.

Technologies: ASP.NET, Visual Basic, SQL Server

INSTALLATION:

1. Attach the database files (.mdf) by using SQL Server Management Studio Express

2. Upload the root directory files

3. Update the database connection settings on DBConnection.vb file

4. Login Details: username- rasan password- 1234
